Pro Tips for Using CORE

Tip 1: Start Small, Build Up

You don't always need all four CORE elements. Start with Context and Output (the essentials), then add Rules and Examples as needed for more control.

Minimum viable prompt: Context + Output
Professional prompt: All four CORE elements

Tip 2: Create a CORE Template

Save time by creating templates for common tasks. Fill in the blanks each time you need that type of prompt.

Email template:
C: "You are [role] writing to [audience]..."
O: "Write [length] email with [structure]..."
R: "Use [tone], avoid [constraints]..."
E: "Like this: [sample opening]..."

Tip 3: Iterate and Refine

Your first CORE prompt won't always be perfect. Test it, see the results, then refine the elements that need improvement.

Tip 4: Examples are Powerful

When you're struggling to describe the style you want, don't spend 10 minutes writing rules—just show a quick example instead. Examples often work better than lengthy descriptions.

Tip 5: Teach Others

The best way to solidify your understanding of CORE is to teach it to someone else. Show a colleague how to improve their prompts using the framework.

CORE Framework Quick Reference

C - Context

Setting the Stage

Background, role, situation, and who you're addressing

O - Output

Defining the Result

Format, length, structure, and organization

R - Rules

Setting Boundaries

Tone, constraints, what to avoid, and guidelines

E - Examples

Showing the Standard

Sample phrases, style demonstrations, and models
